# Plataforma de Gamificação com Doações

Este projeto consiste em uma plataforma web de gamificação criada para incentivar o engajamento de usuários por meio de desafios, sistema de pontos e ranking.  
A aplicação também integra um fluxo de doações, conectando a participação dos usuários a ações sociais.

O sistema foi desenvolvido em grupo como projeto apresentado na Expotec, com foco em aprendizado prático de desenvolvimento web e colaboração em equipe.

---

## Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- PHP  
- MySQL  

---

## Funcionalidades
- Sistema de pontuação e lógica de gamificação  
- Ranking de usuários  
- Cadastro e autenticação  
- Interface web responsiva  
- Integração com fluxo de doações  
- Jogos integrados à plataforma de gamificação  

---

## Minha Contribuição
Minhas principais contribuições neste projeto incluem:
- Desenvolvimento do front-end das páginas principais da plataforma  
- Implementação das regras de pontuação e da lógica de gamificação  
- Criação da interface de ranking de usuários  
- Desenvolvimento do jogo **"Aviador"**, sendo minha responsabilidade a lógica, a interface e a integração com o sistema de pontuação da plataforma  
- Apoio na integração das funcionalidades relacionadas ao fluxo de doações  

---

## Como Executar o Projeto
1. Clone este repositório  
2. Configure o banco de dados MySQL  
3. Inicie um servidor local (ex: XAMPP ou WAMP)  
4. Acesse o projeto pelo navegador
